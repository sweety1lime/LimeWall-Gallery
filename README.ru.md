# LimeWall

[English](README.md) · **Русский**

Живые обои для рабочего стола, от 2fame. Сначала Windows (Linux и macOS —
позже). Проигрывает видео / GIF (→ mp4) / картинки / HTML / 3D (glTF) позади
иконок рабочего стола с минимальной нагрузкой и апскейлом на видеокарте.

> ⚠️ Проект в активной разработке. Сейчас это **бета (0.1.0-beta)** — что-то
> может работать неидеально. Отчёты и идеи очень помогают.

## Что это

Приложение живых обоев: ставит видео, картинку, веб- или 3D-сцену как фон
рабочего стола. Главный приоритет — **минимальная нагрузка на систему** (≈0% CPU,
когда обои на паузе). Ядро (плеер + импорт своих файлов) — бесплатно. Похожие
программы: Wallpaper Engine (платная) и Lively (бесплатная).

Внутри — два процесса: фоновый плеер `renderer` (работает всегда, значок в трее)
и окно управления. Окно можно закрывать — обои продолжат играть.

## Что в этом репозитории

Не пугайся количества папок — **обычному пользователю нужен только готовый билд**
(см. «Установка»). Всё остальное — это исходный код:

- `crates/` — движок на Rust (плеер, платформенный слой, формат пакетов);
- `apps/ui/` — окно управления (Tauri + TypeScript);
- `gallery/` — общий каталог обоев и его проверки;
- `scripts/` — скрипты сборки и загрузки зависимостей;
- `docs/`, `PLAN.md` — заметки, решения и роадмап.

## Возможности

- **Видео, GIF и картинки** через libmpv с аппаратным декодированием (d3d11va);
  GIF при импорте превращается в mp4.
- **Веб (HTML) и 3D (glTF)** через WebView2, для моделей — встроенный
  three.js-вьювер.
- **Апскейл на GPU** — lanczos + FSR, либо Anime4K для рисованного контента;
  включается, только если исходник меньше монитора.
- **Настройки на каждый монитор** (качество, громкость, Anime4K) и **плейлисты**
  (интервал + перемешивание), которые крутятся даже с закрытым окном.
- **Экономия ресурсов** — декодирование встаёт на паузу (≈0% CPU) под
  полноэкранными приложениями, на экране блокировки, при выключенном дисплее и
  от батареи (пауза / эко / без изменений). Отдельный «сторож» паузит обои,
  которые греют процессор.
- **Защита от недоверенного контента** — веб-страницам выдаётся строгий CSP (без
  выхода в сеть и удалённого кода), а установка код-обоев (web/3D) спрашивает
  согласие. Подробнее: [docs/research/security-model.md](docs/research/security-model.md).
- **Пакеты `.wpk`** (zip + манифест) — экспорт любого элемента библиотеки, импорт
  через диалог, перетаскиванием или двойным кликом.
- **Общий каталог** — можно смотреть и ставить чужие обои прямо из вкладки
  «Каталог»; загрузки проверяются по контрольной сумме. См.
  [gallery/README.md](gallery/README.md).

## Установка

Проще всего — **портейбл-сборка** (всё в одной папке, без установщика):

1. Возьми готовый архив из раздела Releases (когда будет) — или собери сам (ниже).
2. Распакуй папку целиком и запусти `LimeWall.exe`.
3. Внутри папки лежит `README.txt` с короткой инструкцией для тестеров.

## Сборка из исходников

Один раз — скачать проигрыватель и лицензии (запиненные LGPL-сборки; см.
[docs/third-party.md](docs/third-party.md)):

```
scripts\fetch-libmpv.ps1
scripts\fetch-ffmpeg.ps1
scripts\fetch-licenses.ps1
```

Собрать портейбл-папку:

```
scripts\build-portable.ps1
```

Результат — в `dist/LimeWall/`; запускать `dist\LimeWall\LimeWall.exe`.

Для разработки есть режим окна и CLI:

```
cargo run -p renderer -- play video.mp4 --monitor 0 --quality max
cargo run -p renderer -- serve
cd apps/ui && npm install && npm run tauri dev
```

> В Windows PowerShell 5 нет `&&` — вводи команды по одной. Первый `tauri dev`
> компилирует весь стек Tauri и может занять несколько минут до появления окна.

## Безопасность и приватность

- Веб-обои изолированы: строгий CSP — **без доступа в интернет** и без загрузки
  удалённого кода.
- Установка код-обоев (web/3D) требует явного согласия.
- Обои из каталога проверяются по **SHA-256**; вредоносный пак можно **отозвать**
  (kill-switch удаляет его у всех, кто уже скачал).
- **Без телеметрии.** В сеть приложение ходит только за каталогом (наш GitHub) и
  только если ты пользуешься галереей.

## Свои обои в каталог

Каталог принимает **только видео и картинки** (через Pull Request с модерацией) —
интерактивные web/3D пока не принимаются, их нельзя безопасно проверить
автоматически. Как отправить свой пак: [gallery/README.md](gallery/README.md).

## Лицензии

Ядро (плеер + локальный импорт) — бесплатно. mpv и ffmpeg используются как
**LGPL**-сборки с динамической линковкой; шейдеры (FSR / Anime4K) — MIT/BSD.
Источники зафиксированы в [docs/](docs/), а в готовой сборке лежит папка
`licenses/` с полными текстами.
